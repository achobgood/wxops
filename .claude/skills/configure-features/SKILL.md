## When to Invoke

After confirming the user wants to set up call features: auto attendants, call queues, hunt groups, paging groups, call park, call pickup, voicemail groups, or CX Essentials.

## Workflow

1. Verify auth token is working
2. Identify which feature(s) to configure
3. Gather feature-specific requirements
4. Build deployment plan
5. Execute API calls
6. Verify results
